# node-toolchain

[![npm version](https://badge.fury.io/js/%40twihike%2Fnode-toolchain.svg)](https://badge.fury.io/js/%40twihike%2Fnode-toolchain)

## Installation

With npm:

```shell
npm install --save-dev @twihike/node-toolchain
```

With yarn:

```shell
yarn add --dev @twihike/node-toolchain
```
