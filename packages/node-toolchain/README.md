# node-toolchain

[![npm version](https://badge.fury.io/js/%40tkhiking%2Fnode-toolchain.svg)](https://badge.fury.io/js/%40tkhiking%2Fnode-toolchain)

## Installation

With npm:

```shell
npm install --save-dev @tkhiking/node-toolchain
```

With yarn:

```shell
yarn add --dev @tkhiking/node-toolchain
```
